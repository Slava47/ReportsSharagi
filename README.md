# CodeLab Assistant

Автоматический генератор отчетов для лабораторных работ по программированию.

## Описание

CodeLab Assistant — это программа, которая автоматически создает готовые отчеты для лабораторных работ по программированию. Предназначена для студентов, которые изучают **Pascal**, **Python** и **C++**.

Программа принимает на вход файл с исходным кодом и набор тестовых данных, а затем автоматически:

- Анализирует код: определяет язык, извлекает комментарии, определяет алгоритмы
- Компилирует и запускает программу с тестовыми данными
- Строит блок-схему алгоритма
- Создает отчет в формате Word (.docx)

Готовый отчет содержит:

- Титульный лист с данными студента
- Цель работы
- Блок-схему алгоритма
- Листинг программы с подсветкой синтаксиса
- Результаты тестирования в таблице
- Выводы

## Установка

```bash
pip install -r requirements.txt
```

Для генерации блок-схем также необходимо установить [Graphviz](https://graphviz.org/download/).

## Использование

### Графический интерфейс (GUI)

```bash
python -m codelab_assistant
```

Откроется окно, где нужно:
1. Выбрать файл с исходным кодом
2. Ввести данные студента (ФИО, группа, вариант)
3. Ввести тестовые данные
4. Нажать «Сгенерировать отчет»

### Командная строка (CLI)

```bash
python -m codelab_assistant generate program.py \
  --name "Иванов Иван Иванович" \
  --group "ИТ-21" \
  --variant "5" \
  --test-data "1 2 3" "4 5 6"
```

Тестовые данные можно также загрузить из файла:

```bash
python -m codelab_assistant generate program.py \
  --test-file tests.txt \
  --output report.docx
```

Формат файла с тестами — тесты разделяются строкой `---`:

```
1 2 3
---
4 5 6
---
10 20 30
```

### Управление профилями преподавателей

```bash
# Список профилей
python -m codelab_assistant profiles list

# Показать настройки профиля
python -m codelab_assistant profiles show --name default

# Создать новый профиль
python -m codelab_assistant profiles create --name my_teacher
```

## Поддерживаемые языки

| Язык    | Расширения        | Компилятор/Интерпретатор |
|---------|-------------------|--------------------------|
| Pascal  | `.pas`            | PascalABC, Free Pascal   |
| Python  | `.py`             | Python 3                 |
| C++     | `.cpp`, `.cc`, `.c` | g++                    |

## Зависимости

- **python-docx** — создание Word документов
- **Pygments** — подсветка синтаксиса кода
- **graphviz** — создание блок-схем
- **Pillow** — работа с изображениями

## Структура проекта

```
codelab_assistant/
├── __init__.py          # Инициализация пакета
├── __main__.py          # Точка входа
├── analyzer.py          # Анализ исходного кода
├── executor.py          # Компиляция и запуск
├── flowchart.py         # Генерация блок-схем
├── report_generator.py  # Генерация Word отчетов
├── profiles.py          # Управление профилями
├── cli.py               # Интерфейс командной строки
└── gui.py               # Графический интерфейс
```

## Лицензия

Программа бесплатная и с открытым исходным кодом.
